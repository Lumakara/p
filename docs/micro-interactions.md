# Micro Interactions

## Tujuan
Membuat UI terasa hidup, responsif, dan premium tanpa berlebihan.

## Interaksi utama
### Button
- hover: sedikit berubah
- active: sedikit mengecil
- disabled: jelas dan konsisten

### Product card
- hover: elevasi sedikit naik
- image: boleh zoom sangat halus
- wishlist: dapat feedback visual
- add to cart: konfirmasi kecil

### Search
- focus state jelas
- suggestion state jika ada
- loading state ringan saat query diproses

### Cart
- quantity update harus terasa instan
- subtotal berubah dengan feedback yang halus
- item removal harus punya konfirmasi yang jelas

### Checkout
- step transition harus sederhana
- validasi harus jelas
- error state harus informatif

## Principles
- Feedback cepat
- Tidak mengganggu
- Konsisten di semua halaman
- Selalu punya alasan fungsi
