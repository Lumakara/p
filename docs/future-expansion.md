# Future Expansion

## Roadmap kemungkinan
### Phase 1
- Frontend core
- Homepage
- Listing
- Detail
- Cart
- Checkout

### Phase 2
- User account
- Wishlist
- Order history
- Search improvement
- Promo engine

### Phase 3
- Seller/admin modules
- Analytics dashboard
- Mobile app adaptation
- Advanced personalization

## Scalability guidelines
- Design system harus tetap konsisten.
- Component library harus bertambah tanpa merusak sistem.
- Tokens harus menjadi pusat perubahan visual.
- Referensi baru harus diintegrasikan dengan aturan yang sama.

## Long-term goal
Menghasilkan frontend system yang cukup matang untuk dipakai ulang di web, app, dan dashboard tanpa rework besar.
