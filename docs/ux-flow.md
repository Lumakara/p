# UX Flow

## Core user flow
1. User masuk ke homepage.
2. User langsung memahami kategori dan produk unggulan.
3. User mencari atau membuka kategori.
4. User melihat detail produk.
5. User menambahkan ke cart.
6. User checkout dengan proses singkat.
7. User menerima success state yang jelas.

## Product type handling
### Produk fisik
- cek alamat
- cek ongkir
- cek pengiriman
- summary total

### Produk digital
- jelaskan bentuk produk
- jelaskan delivery method
- tampilkan akses/pengunduhan setelah pembayaran

## UX rules
- Hindari langkah tak perlu.
- Hindari teks yang terlalu teknis.
- Hindari dialog yang menghalangi alur tanpa alasan.
- Buat CTA selalu terlihat.
