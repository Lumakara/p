# Accessibility

## Target
Minimal aksesibilitas dasar harus baik dan tidak mengganggu pengalaman.

## Rules
- Kontras teks harus cukup.
- Semua button harus punya focus state.
- Semua input harus punya label yang jelas.
- Alt text wajib untuk gambar penting.
- Interaksi tidak boleh hanya mengandalkan warna.
- Ukuran target sentuh harus nyaman di mobile.
- Navigasi keyboard harus masuk akal.

## Content clarity
- Teks harus mudah dipindai.
- CTA harus jelas.
- Error message harus menjelaskan masalah dan tindakan berikutnya.

## Motion safety
- Animasi tidak boleh berlebihan.
- Bila user punya preferensi reduced motion, pengalaman harus tetap nyaman.

## UX accessibility
- Jangan mengunci user dalam modal tanpa jalan keluar jelas.
- Jangan membuat filter terlalu rumit.
- Jangan menyembunyikan informasi penting di tempat yang sulit ditemukan.
