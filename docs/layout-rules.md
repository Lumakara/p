# Layout Rules

## Prinsip umum
Layout harus terasa terarah, lega, dan mudah dipindai mata.

## Hierarki visual
1. Navbar / search
2. Hero / promosi utama
3. Kategori
4. Promo cards
5. Product grids
6. Trust section
7. Footer

## Grid philosophy
- Gunakan grid yang konsisten.
- Jaga alignment.
- Hindari section yang berdiri tanpa ritme.
- Desktop harus terasa kaya tetapi tidak berantakan.
- Mobile harus memadat secara elegan, bukan sekadar mengecil.

## Spacing
- Jarak antar section harus jelas.
- Card punya ruang napas internal.
- Jangan menekan banyak informasi dalam satu area.
- Gunakan spacing sebagai penanda prioritas.

## Surface hierarchy
- Background utama
- Section surface
- Card surface
- Floating elements
- Modal / drawer

## Sidebar
Jika ada sidebar:
- harus jelas,
- mudah dipindai,
- tidak terlalu ramai,
- dan bisa collapse di layar kecil.

## Header
Header harus membantu pencarian dan navigasi, bukan mengalihkan perhatian.

## Hero
Hero wajib:
- singkat,
- fokus,
- punya CTA jelas,
- dan memberi identitas visual kuat.

## Card composition
Card harus:
- rapi,
- mudah dipahami,
- punya image yang dominan,
- metadata yang cukup,
- dan CTA yang tidak memaksa.

## Dense data handling
Kalau ada banyak item:
- gunakan grouping,
- sectioning,
- sorting,
- filtering,
- dan progressive disclosure.
