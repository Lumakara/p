# Responsive System

## Filosofi
Responsive bukan hanya mengecilkan layout. Responsive berarti setiap device mendapat pengalaman terbaik sesuai konteksnya.

## Breakpoint concept
- Mobile
- Small tablet
- Tablet
- Laptop
- Desktop
- Wide desktop

## Mobile behavior
- Prioritaskan scroll vertikal
- Sidebar berubah menjadi drawer/bottom sheet
- CTA penting tetap terlihat
- Filter harus mudah diakses
- Navigasi harus sederhana

## Tablet behavior
- Layout bisa dua kolom
- Komponen tetap lega
- Tidak terlalu padat
- Cart/filter bisa menjadi panel

## Desktop behavior
- Sidebar boleh permanen
- Grid produk bisa lebih kaya
- Cart panel bisa tampil jelas
- Ruang visual bisa lebih premium

## App / mobile-app feel
Jika desain dipakai untuk app:
- gunakan bottom navigation bila relevan,
- gunakan drawer untuk panel berat,
- gunakan modal secukupnya,
- hindari terlalu banyak floating element.

## Rules
- Tidak ada elemen penting yang hilang di mobile.
- Tidak ada teks yang terlalu kecil.
- Tidak ada komponen yang hanya bagus di desktop.
- Semua interaksi harus tetap dapat dipahami di layar kecil.
