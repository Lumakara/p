# Component Library

## Aturan umum
Semua komponen harus reusable, konsisten, dan mengikuti design tokens.

## Komponen inti
### 1. Button
Variants:
- primary
- secondary
- tertiary
- ghost
- destructive
States:
- default
- hover
- active
- disabled
- loading

### 2. Card
Variants:
- product card
- promo card
- info card
- cart item card
- feature card

### 3. Badge
Digunakan untuk:
- promo
- status
- category
- new
- hot
- stock

### 4. Input
States:
- default
- focus
- error
- disabled
- loading

### 5. Search bar
- ikon jelas
- placeholder informatif
- hasil cepat ditemukan
- mudah digunakan di mobile

### 6. Sidebar / Nav
- readable
- tidak terlalu ramai
- active state jelas
- collapse-friendly

### 7. Product Grid
- rapi
- responsif
- image dominan
- metadata cukup

### 8. Product Detail
Harus mendukung:
- gallery
- price
- variant
- stock
- CTA
- detail digital/fisik

### 9. Cart Drawer / Cart Panel
Harus menampilkan:
- item list
- qty
- summary
- promo code
- checkout CTA

### 10. Checkout Modules
- shipping info
- digital delivery info
- payment summary
- success state

### 11. Skeleton
Gunakan untuk:
- list loading
- product cards
- detail page
- cart loading

### 12. Empty State
Buat untuk:
- cart kosong
- wishlist kosong
- search kosong
- order kosong

### 13. Toast / Snackbar
Dipakai untuk:
- add to cart
- wishlist update
- promo applied
- error ringan
- success ringan

### 14. Modal / Drawer
Dipakai untuk:
- filter
- auth
- quick view
- cart
- confirm actions

## Design rules for components
- Semua radius harus konsisten.
- Semua shadow harus terasa satu keluarga.
- Semua icon harus satu style.
- Semua spacing harus mengikuti sistem.
- Semua komponen harus mudah dirakit ulang.
