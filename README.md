# Claude Code Frontend Bible

Ini paket dokumentasi untuk project e-commerce hybrid (produk digital + fisik) dengan arah visual premium, modern, unik, cepat, dan mudah dipakai.

Gunakan file berikut sebagai acuan utama:
- `CLAUDE.md`
- `PRD/PRODUCT_REQUIREMENTS_DOCUMENT.md`
- `docs/design-philosophy.md`
- `docs/reference-policy.md`
- `docs/color-system.md`
- `docs/typography.md`
- `docs/layout-rules.md`
- `docs/animation-system.md`
- `docs/component-library.md`
- `docs/responsive-system.md`
- `docs/coding-standards.md`
- `docs/design-constraints.md`

Folder `reference/` disediakan untuk menyimpan gambar referensi yang akan disisipkan manual nanti.
