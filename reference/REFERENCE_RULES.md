# Reference Rules

## Purpose
Folder reference berisi gambar inspirasi untuk membantu memahami layout, hierarchy, komposisi, dan rasa visual.

## Aturan penting
- Jangan copy mentah.
- Jangan mempertahankan identitas warna referensi jika bertentangan dengan sistem warna proyek.
- Gunakan gambar sebagai petunjuk struktur, bukan sebagai template final.
- Semua identitas brand final harus dibangun ulang.
- Bila ada konflik, dokumen proyek tetap menang.

## Cara pakai
Simpan gambar sesuai fungsinya:
- home
- listing
- product
- cart
- checkout
- mobile
- animation
- interaction

## Catatan
Referensi terbaik adalah referensi yang menjelaskan tujuan visual, bukan sekadar gambar yang terlihat bagus.
